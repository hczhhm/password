<div class="title"><h2>隐私政策</h2><p style="text-align:center;color:#999;font-size:0.75rem">最后更新日期：2026年6月5日</p></div>
<div class="body">

<p>密码管理（以下简称"本软件"）高度重视用户隐私保护。本政策依据《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《中华人民共和国数据安全法》制定。</p>

<div class="high"><strong>核心承诺：</strong>本软件采用"本地优先·零知识"架构。所有密码数据以 AES-256 加密后存储在您的设备本地。我们不收集、不上传、不访问您的密码数据和个人身份信息，也无法解密您的数据。您对自己的数据拥有完全的控制权。</div>

<h3>一、不收集的信息</h3>
<p>本软件<strong>不会</strong>收集或传输以下任何信息：</p>
<ul>
<li><strong>密码数据</strong>：以密文存储在设备本地 SQLite 数据库，加密密钥由主密码派生，不可被我们解密。</li>
<li><strong>主密码</strong>：仅以哈希值存储在本地，用于登录验证，永不离设备。</li>
<li><strong>密保答案</strong>：以哈希值不可逆存储，仅用于身份验证。</li>
<li><strong>个人身份信息</strong>：不需注册，不收集姓名、手机号、身份证号、邮箱地址等。</li>
<li><strong>使用分析数据</strong>：无第三方分析、广告、追踪 SDK。</li>
</ul>

<h3>二、系统服务涉及的数据传输</h3>
<p>2.1 <strong>云端备份</strong>：加密备份文件通过设备系统云服务同步，数据安全性由对应平台保障。本软件不读取传输内容。</p>
<p>2.2 <strong>生物识别</strong>：指纹/面部识别由操作系统安全区域完成，本软件仅接收验证结果，无法获取原始生物特征数据。</p>

<h3>三、数据安全措施</h3>
<p>3.1 <strong>加密算法</strong>：AES-256-CBC 加密存储。密钥通过 MD5（主密码+固定盐值）派生。</p>
<p>3.2 <strong>备份加密</strong>：独立 AES-256 加密，备份密码与主密码分离。</p>
<p>3.3 <strong>安全建议</strong>：使用高强度主密码（8位以上，含大小写字母、数字、特殊字符），开启设备锁屏，定期备份。</p>

<h3>四、您的权利（依据《个人信息保护法》）</h3>
<ul>
<li><strong>知情权与访问权</strong>：随时查看所有密码数据。</li>
<li><strong>更正权</strong>：随时编辑修改。</li>
<li><strong>删除权</strong>：随时删除或彻底清除。卸载即删除全部本地数据。</li>
<li><strong>数据可携权</strong>：通过加密备份文件随时导出迁移。</li>
<li><strong>撤回同意权</strong>：卸载软件即视为撤回同意。</li>
</ul>
